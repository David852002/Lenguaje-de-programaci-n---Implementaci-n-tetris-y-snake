Guía de Programación del Lenguaje Brik

Referencia oficial de sintaxis para el motor de juegos BrikVM.

1. Sintaxis Básica

Punto y Coma (;)

Toda sentencia en Brik debe terminar con un punto y coma. Esto indica el fin de una instrucción.

print("Hola");
let x = 10;

Comentarios

Los comentarios son de una sola línea y comienzan con //. El intérprete los ignora.

// Esto es un comentario.
let score = 0; // Asigna el puntaje inicial

2. Variables y Tipos de Datos

Declaración de Variables (let y const)

Las variables deben ser declaradas usando let o const. Brik utiliza ámbito de función (las variables son visibles en toda la función donde se declaran, similar a var en JavaScript antiguo).

// 'let' permite re-asignar el valor
let puntuacion = 100;
puntuacion = 150;

// 'const' define una constante (aunque el runtime actual no fuerza la inmutabilidad)
const ALTO = 20;

Tipos Primitivos

  • Number (Número): Todos los números se manejan como punto flotante (float). 
    let edad = 25;
let precio = 10.5;
  • String (Cadena): Texto definido con comillas dobles. 
    let nombre = "Snake";
  • Boolean (Booleano): Los valores true y false están disponibles globalmente. 
    let gameOver = false;

Arrays (Listas)

Brik soporta arrays anidados, esenciales para los tableros de juego.

Creación

let miArray = [1, "hola", true];
let matriz = [[1, 2], [3, 4]];

Lectura (Acceso por Índice)

print(miArray[1]);     // Imprime "hola"
print(matriz[0][1]); // Imprime 2.0

Escritura (Asignación por Índice)

miArray[0] = 99;
matriz[1][1] = 0;

3. Operadores

Aritméticos

Manejo especial de +: si uno de los operandos es un string, Brik forzará la concatenación de strings.

print(5 + 10);     // Imprime 15.0
print("Score: " + 100); // Imprime "Score: 100.0"

Operadores disponibles: +, -, *, /, % (módulo/resto).

Lógicos y de Comparación

Se usan principalmente en sentencias if y while.

  • Comparación: == (igual), != (no igual), >, <, >=, <=
  • Lógicos: && (Y), || (O)
if (score > 100 && isPaused == false) {
    // ...
}

Nota: El operador ! (NOT) no está implementado. Para invertir un booleano, usa una sentencia if/else.

Asignación

El operador = se usa para asignar valores a variables, propiedades de clase o índices de array.

let x = 10;
x = 20;
self.propiedad = 5;
miArray[0] = 1;

4. Flujo de Control

if / else if / else

Ejecuta bloques de código condicionalmente. Las llaves {...} son obligatorias.

if (tecla == "a") {
    direccion = constantes.IZQUIERDA;
} else if (tecla == "d") {
    direccion = constantes.DERECHA;
} else {
    // No hacer nada
}

while

Ejecuta un bloque de código repetidamente mientras la condición sea true.

let i = 0;
while (i < 5) {
    print(i);
    i = i + 1;
}

break

Termina la ejecución del bucle while más cercano.

while (true) {
    if (gameOver == true) {
        break; // Sale del bucle
    }
}

5. Funciones y Clases (OOP)

Brik es un lenguaje orientado a objetos basado en clases.

Funciones Globales (def)

Puedes declarar funciones en el ámbito global para reutilizar lógica.

def sumar(a, b) {
    return a + b;
}
let resultado = sumar(5, 3);
print(resultado); // Imprime 8.0

return

La sentencia return devuelve un valor de una función y termina su ejecución. Si no se usa, la función devuelve un valor nulo (None).

Clases (class)

Define una "plantilla" para un objeto. Las propiedades se definen usando let.

class Constantes {
    let ALTO = 12;
    let ANCHO = 16;
}

Métodos y self

Los métodos son funciones definidas dentro de una clase. El primer parámetro **debe ser self** para acceder a los atributos de la instancia.

class Auto {
    let marca = "Brik";

    def getMarca(self) {
        return self.marca;
    }

    def setMarca(self, nuevaMarca) {
        self.marca = nuevaMarca;
    }
}

Instanciación (new)

Crea un objeto (instancia) a partir de una clase.

let miAuto = new Auto;

Acceso a Miembros (.)

Se usa el punto para leer atributos y llamar a métodos.

print(miAuto.marca); // Lee el atributo
miAuto.setMarca("Tesla"); // Llama al método
print(miAuto.getMarca());

6. API Nativa (Funciones Incorporadas)

El entorno de Brik proporciona un conjunto de funciones esenciales (implementadas en Python) para interactuar con el sistema.

Función (en Brik) Descripción Funcional Ejemplo de Uso
print(valor) Imprime un valor en la consola, seguido de un salto de línea. print("Score: " + score);
clear_screen() Limpia toda la pantalla de la consola. clear_screen();
sleep(ms) Pausa la ejecución del juego por un número de milisegundos. sleep(100); // Pausa 100ms
get_key() Comprueba si se ha presionado una tecla. Devuelve el carácter (ej. "w") o None si no se presionó nada. No bloquea el juego. let tecla = get_key();
len(arr_o_string) Devuelve la longitud (número) de un array o un string. while(i < len(serpiente))
rand(min, max) Devuelve un número entero aleatorio entre min y max (ambos incluidos). comida = [rand(0, 19), rand(0, 9)];
push_front(arr, val) Inserta un valor al inicio de un array. push_front(serpiente, nueva_cabeza);
pop_tail(arr) Elimina y devuelve el último elemento de un array. pop_tail(serpiente);
